## get-start

```shell
yarn global add hexo-cli

hexo init myblog
cd myblog
yarn

hexo generate
hexo server
```

### plugin

```shell
yarn add hexo-filter-plantuml
```

### theme

```shell
# theme: next
# or: yarn add hexo-theme-next
git clone https://github.com/next-theme/hexo-theme-next themes/next
```
