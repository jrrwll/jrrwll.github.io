## 用户满意衡量

- 净推荐值 NPS
NPS（Net Promoter Score），净推荐值，又称净促进者得分，亦可称口碑，是一种计量某个客户将会向其他人推荐某个产品或服务可能性的指数。

Promoter 推荐者，满意且推荐，十分制取9-10分
Passive 中立者，满意但没有推荐，十分制取7-8分
Detractor 贬低者，不满意，十分制取0-6分

`NPS = PromoterCnt / TotalCnt - DetractorCnt / TotalCnt`

NPS <= 0：需要改善
0% < NPS <= 30%：良好，但仍有进步的空间
30% < NPS <= 70%：优秀，客户的满意程度较高。
70% < NPS <= 100%：非常优秀，产品可以通过客户的推荐积攒良好的口碑

- 用户满意度 CSAT
用户满意度一般是用来衡量用户对一款产品服务和体验的整体满意程度。

使用五点量表，包括五个选择：5=非常满意、4=满意、3=一般、2=不满意、1=非常不满意。计算选择 4-5 分的用户所占比例，得到最终的 CSAT 值。

`CSAT = ( 4Cnt + 5Cnt ) / TotalCnt`

- 用户费力度 CES
让用户评价使用某产品/服务来解决问题的困难程度。

采用7分制给出对此表述的同意程度，1=强烈不同意、2=不同意、3=有点不同意、4=中立、5=有点同意、6=同意、7=强烈同意。用户费力度取用户给出的费力度评价均值

`CES = AVG(CES1 + CES2 + ... + CESn)`

如何选择衡量方法
- 持续收集用户对产品的评价或进行战略性用户调研 —— NPS
- 针对具体功能点和使用场景进行调研 —— CAST
- 需要评估产品功能/交互是否好用，会否增加使用成本 —— CES
